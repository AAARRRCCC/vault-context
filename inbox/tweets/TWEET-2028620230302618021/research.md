# Creative Obsidian vault visualization: notes rendered as a botanical garden with a chronological timeline scrubber

## Substance

This tweet is a brief endorsement by @kepano (the maker of Obsidian) of a creative visualization project built by @poetengineer__. The project renders an Obsidian vault as an animated "digital garden" — each plant represents a tag, with notes positioned along the plant's structure based on age: older notes appear on the trunk, newer notes appear as leaves. A timeline scrubber allows the user to watch the vault "grow" chronologically, simulating the experience of tending a living garden.

The core concept is an aesthetic/metaphorical interface layer on top of a standard Obsidian vault. Rather than a graph view, a kanban, or a list, the visualization leans into a biological growth metaphor as a way to make note-taking feel more organic and personally meaningful. The timeline interaction is the standout mechanic — it transforms a static graph into something temporal and narrative.

@kepano's comment ("we're barely scratching the surface") signals that Obsidian's team sees this as an example of unexplored design space in vault visualization. It reads as both appreciation for the project and a soft design manifesto: the canonical graph view is not the ceiling.

No GitHub repo, blog post, or external link was included in the tweet. This is a demo/aesthetic showcase, not a released tool.
