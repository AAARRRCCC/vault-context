# PQO Icons (pqoqubbw) — open-source animated icon library built on Motion and Lucide

## Substance

PQO Icons (also referred to as pqoqubbw Icons) is a free, open-source collection of animated SVG icons designed for use in web and app projects. The icons are built on top of two well-known libraries: **Motion** (formerly Framer Motion, the React animation library) and **Lucide** (a popular fork of Feather Icons). The combination means each icon ships as a React component with smooth, pre-built entry/interaction animations, while the underlying vector shapes come from Lucide's clean, consistent design system.

The library is positioned squarely at developers and designers who want to add polish to a React-based UI without hand-authoring animation code. Icons are customizable — size, color, and presumably animation trigger/duration can be adjusted — and because the base shapes are Lucide-derived, there's a large vocabulary of icons covering common UI needs (navigation, actions, status indicators, etc.).

The project is open-source (MIT or similar permissive license is implied by the "free" framing), which means it can be dropped into any project without licensing overhead. It is not a SaaS product or a paid tier tool — it's a static component library.
